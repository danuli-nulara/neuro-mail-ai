# 📧 Neuro-Mail AI

An intelligent email assistant built with BERT and Gemini AI.

## Features
- 🔍 Phishing email detector (fine-tuned BERT)
- ✍️ Email generator (Gemini AI)
- ✅ Email address validator
- 🔗 URL safety scanner

## Tech Stack
- BERT (fine-tuned for phishing detection)
- Gemini AI (email generation)
- Gradio (web interface)
- Hugging Face Spaces (deployment)
- Google Safe Browsing API (URL scanning)

## Model
Trained on a phishing email dataset with:
- Accuracy: 98%+
- Fine-tuned BERT base uncased
- 1500 samples, 5 epochs
